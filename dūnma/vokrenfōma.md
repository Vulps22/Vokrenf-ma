# Etymology of vokrenfōma

## English Translation
- language (Vokrenfōma)

## Word Type
- dūnma (multi-syllable: vok + ren + fō + ma)

## Breakdown (for dūnma only)
- vok (speak) + ren (unknown) + fō (manner) + ma (group)
