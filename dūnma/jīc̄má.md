# Etymology of jīc̄má

## IPA
- /dʒɪx.mja/

## English Translation
- upper limb, arm (up-extend)

## Word Type
- dūnma (multi-syllable: jīc̄ + má)

## Breakdown (for dūnma only)
- jīc̄ (up) + má (extend/grow)

## Justification
- An upward extension from the body. Used in body phrases: jīc̄má c̄erem = neck; retne jīc̄má c̄erem = head; ja jīc̄má c̄erem = breast.
