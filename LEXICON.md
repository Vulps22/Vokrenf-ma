# Body

Body part expressions are built compositionally from directional dŇęn, **m√°** (extend), **cŐĄerem** (body), and **retne** (extremity). The phrase structure is: `[modifier(s)] cŐĄerem`.

| Phrase | Literal | Meaning |
| :---- | :---- | :---- |
| cŐĄerem | core(person) | torso, body |
| jńęcŐĄ cŐĄerem | up body | chest |
| nacŐĄ cŐĄerem | down body | abdomen, stomach |
| jńęcŐĄm√° cŐĄerem | up-extend body | neck |
| nacŐĄm√° cŐĄerem | down-extend body | leg |
| jocŐĄm√° cŐĄerem | left-extend body | left arm |
| mocŐĄm√° cŐĄerem | right-extend body | right arm |
| retne jńęcŐĄm√° cŐĄerem | end-thing of up-limb | head |
| retne nacŐĄm√° cŐĄerem | end-thing of down-limb | foot |
| ja nacŐĄm√° cŐĄerem | front of down-limb | penis |
| ja jńęcŐĄm√° cŐĄerem | front of up-limb | breast |

# Senses

Sense words share a **v-** onset, an emergent phonological grouping alongside **vid** (see). Sense organs are formed as **v-sense + ret** (thing).

| Phrase | Literal | Meaning |
| :---- | :---- | :---- |
| vidret | see(thing) | eye |
| vekret | hear(thing) | ear |
| vńękret | smell(thing) | nose |
| vatret | taste(thing) | mouth, tongue |
| vonret | touch(thing) | skin |
