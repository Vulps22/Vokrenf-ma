# Etymology of má

## IPA
- /mja/

## English Translation
- extend, grow

## Word Type
- dūn (single syllable)

## Breakdown (for dūnma only)
- N/A

## Justification
- Uses the vowel á /ja/, which has been in the language since ná. Used to form limb/appendage compounds (e.g. jīc̄má = upper limb, nac̄má = lower limb).
